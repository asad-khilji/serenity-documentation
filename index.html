How to Edit the Home Page

Navigate to your store url / admin for the admin panel

Go to the themes section on the left side panel

Click on customize

Here you can choose to show Collections or Featured products

Here you can update the home page

How to add or remove sections

Navigate to the admin panel and click on themes

Click on the three dots and click edit code

Go to templates and edit the index.liquid to make changes to the home page

You can add or remove the sections you want

To edit the sections go to the sections and click the section you want to edit

How to Edit the Contact page

Navigate to the admin panel on the left side

Click on themes then pages

Select the page you want to edit

To make changes on the backend navigate to the themes and edit code

Under templates click on page-contact.liquid to edit the contact page

How to edit the product page

Navigate to the themes section in the admin panel

Click on the three dots and select edit code

Under templates click on product.liquid to edit the product page

How to Add or Change the styling of the website

Navigate to the admin panel and click themes on the left side

Click on customize then click theme settings

Here you can change the background color or font type

To change more styling go to the themes from admin panel

Click on the three dots then click edit code

Under the assets folder select the css folder you want to change the style of
